Python 學習筆記

請解釋 Python 的 list comprehension 是什麼？

List comprehension 是 Python 的簡潔語法，用來建立清單。

基本語法

squares = [x**2 for x in range(10)]

等同於：

squares = []
for x in range(10):
    squares.append(x**2)

優點

  • 程式碼更簡潔
  • 執行速度通常較快
  • 可讀性高（適合簡單運算）
可以加上條件嗎？

當然可以！使用 if 過濾：

evens = [x for x in range(20) if x % 2 == 0]
